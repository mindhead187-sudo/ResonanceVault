- **Rank structure**: Military titles and designations

- **K-Levels 01-05**: Higher clearance, centralized authority
- **Leader**: Kyra Constantine (Matriarch)

## ğŸ¯ Usage in Queries

### **Find all CMM Executors (K04)**
```sql
SELECT c.character_name, c.codename, ca.military_rank
FROM characters c
JOIN character_corporate_affiliations ca ON c.character_id = ca.character_id
WHERE c.faction = 'CMM' AND ca.clearance_level = '04'
ORDER BY ca.position_title;
```

### **CMM Command Hierarchy**
```sql
SELECT 
    c.character_name,
    c.codename,
    ca.clearance_level,
    ca.military_rank,
    json_extract(c.character_secrets, '$.command_priority') as priority
FROM characters c
JOIN character_corporate_affiliations ca ON c.character_id = ca.character_id
WHERE c.faction = 'CMM'
ORDER BY ca.clearance_level DESC, priority ASC;
```

### **Operator Cell Listing**
```sql
SELECT character_name, codename, primary_role
FROM characters
WHERE faction = 'CMM' AND character_name LIKE 'Unit S-%'
ORDER BY character_name;
```
