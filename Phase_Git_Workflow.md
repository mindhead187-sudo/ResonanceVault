# Phase X Git Workflow

1. Navigate to repo: `cd ResonanceVault`
2. Check status: `git status`
3. Create branch: `git checkout -b phase-X-description`
4. Add files: `git add [files]`
5. Commit: `git commit -m "message"`
6. Push branch: `git push -u origin phase-X-description`
7. Switch to main: `git checkout main`
8. Merge: `git merge phase-X-description`
9. Push main: `git push origin main`
10. Cleanup: `git branch -d phase-X-description`
